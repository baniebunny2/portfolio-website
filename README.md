# CEC Student Portal

A secure and modern login interface designed for Cebu Eastern College students and faculty. This project features a responsive design, glassmorphism UI, and custom branding.

## Features
- **Student ID Login:** Secure authentication for student accounts.
- **Custom UI:** Designed with a professional dark theme consistent with CEC branding.
- **Responsive:** Works seamlessly across desktop and mobile devices.

## Technologies Used
- HTML5
- CSS3 (with custom animations and glassmorphism)
- JavaScript

## Usage
1. Clone this repository.
2. Ensure `index.html`, `CEC.jpg`, and `CECbg.jpg` are in the same folder.
3. Open `index.html` in any web browser to view the login portal.
